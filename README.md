# Wallaby::Her

This gem contains the HER ORM adapter for Wallaby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wallaby-her'
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install wallaby-her
```
